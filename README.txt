Hydra compiler, version 0.1
===============================

This program is free software. You may redistribute it under the terms of
the GNU General Public License version 3 or later. See license.txt for
details.

Included in this release:

   * Lexical analysis

To build, at the terminal type:

    make

To run, type:

    ./hydra.exe <file_name> 
    or
    mono hydra.exe <file_name> 

Where <file_name> is the name of a Hydra source file. You can try with
these files:

   * 001_ultimate.hydra
   * 002_binary.hydra
   * 003_palindrome.hydra
   * 004_factorial.hydra
   * 005_arrays.hydra
   * 006_next_day.hydra
   * 007_literals.hydra
